#MobilFox CURD APP
## Can you see the API project here: https://github.com/FKMLJF/mobilfoxapi
## How to use

1.) `npm install`

2.) run API, and set src/environments/environment.ts api_endpoin="http://yourapi:8081"

3.) `ng serve`


## live demo: https://mobilfox-api.asyncawa.it:8080
